# Análise de Desempenho de Vendas 📊

## Sobre o Projeto
Este projeto consiste em uma Análise Exploratória de Dados (EDA) construída em Python utilizando um dataset de histórico de pedidos. O objetivo principal é extrair insights acionáveis sobre o faturamento, comportamento de compra por região e identificar os produtos mais lucrativos.

## Bibliotecas Utilizadas
* **Pandas & NumPy:** Manipulação, limpeza e agregação de dados.
* **Matplotlib & Seaborn:** Criação de visualizações e gráficos para análise de tendências.

## Principais Insights Extraídos
1. **Efeito Top Performers:** Uma pequena parcela do portfólio de produtos (Top 10) concentra uma grande fatia da receita total (destaque para a copiadora Canon imageCLASS 2200).
2. **Força Tecnológica:** A categoria de **Technology** é o principal motor de faturamento da empresa, superando Furniture e Office Supplies.
3. **Expansão Regional:** A região **Leste (East)** e **Oeste (West)** lideram as vendas, indicando que são os mercados mais fortes, enquanto a região Sul (South) apresenta a menor participação, revelando uma potencial oportunidade de expansão ou necessidade de revisão de estratégia.

## Estrutura do Notebook
1. Carregamento e limpeza de dados (tratamento de valores nulos).
2. Análise Descritiva Estatística.
3. Visualização dos 10 produtos mais vendidos (Gráfico de Barras Horizontais).
4. Análise de distribuição de vendas por categoria (Boxplot).
5. Participação percentual de vendas por região (Gráfico de Pizza).
